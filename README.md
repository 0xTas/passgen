## A simple CLI tool to generate random strings that can be used for passwords or whatever.
<br>

### Usage

Program Defaults: <br>
    - Password Length: 20 <br>
    - Uses Spaces: False <br>

Options: <br>
    - Password Length: -l <br>
    - Use Spaces: -s <br>
    - Command Help: --help <br>
    - Version Info: --version <br>

`./passgen [options]`
